我的Emacs

发表于 2016-07-13

帮助

q 退出, u 前一节 p 下一节
C-h ? 帮助系统的帮助
C-h t help-with-tutorial Emacs 快速指南
C-h r info-emacs-manual Emacs使用手册
C-h i info Emacs说明
C-h a apropos-command 搜索命令
C-h v describe-variable 查看变量说明
C-h f describe-function 查看函数说明
C-h m describe-mode 查看当前mode的相关文档，包含mode中的命令、快捷键
C-h k KEYS describe-key 查看快捷键对应的命令及其简要说明，比如C-h k C-x C-c
C-h c KEYS describe-key-briefly 查看键绑定说明
C-h w where-is 查看某个命令对应的快捷键
C-h b describe-bindings 查看当前buffer所有的快捷键列表
KEYS C-h 查看当前buffer中以某个快捷键序列开头的快捷键列表，比如C-c C-h

窗口

C-z 挂起当前Window (fg 恢复)
C-x 0 关闭当前window
C-x 1 关闭其他window
C-x 2 在下面分割一个window出来
C-x 3 在右边创建一个window
C-x 4 b BUFNAME:在另一个窗口中打开新BUF
C-x 4 f FILENAME:在另一个窗口中打开新文件
C-x 4 d DIRECTORY：
C-x 4 . 在另一个窗口中查找函数定义
C-x 4 r filename：在另一个窗口中打开新文件（只读）

buffer

外观

C-x c-0 调整字体大小

文件

C-x C-f 打开文件
C-x C-v 在当前缓冲区打开另一个文件
C-x C-r 只读打开文件
C-x C-s 保存当前文件
C-x s 保存所有文件
C-x C-w 另存为文件
C-x d dired
C-x C-d 获取文件列表（简洁)
C-x RET r 以指定编码读取文件
C-x RET f 以指定编码保存文件

移动

C-f 向右移动一个字符
M-f 向右移动一个词

C-b 向左移动一个字符 【对中文是移动到下一个标点符号】
M-b 向左移动一个词【对中文是移动到上一个标点符号】

C-n 移动到下一行
C-p 移动到上一行

C-a 移动到行首
C-e 移动到行尾

M-a 移动到句首
M-e 移动到句尾

M-} 移动到上一段
M-{ 移动到下一段

M-> 页尾
M-< 页首

C-M-a 　函数
C-M-e

M-r 页中, 页首, 页尾跳转

M-m 移动到本行第一个非空白字符的位置

跳转

C-x C-x　回到上一个Mark的位置，如可以回到C-s的位置

编辑

撤销重做

C-/ 撤销(每插入20个字符，视为一个 undo 的单位)
C-g C-/ redo
C-x u undo-tree
p 、 n 上下移动
b 、 f 左右切换
t 显示时间戳
d 打开diff，选定需要的状态后
q 退出

删除

删除光标前的一个字符
C-d 删除光标后的一个字符

M- 移除光标前的一个词
M-d 移除光标后的一个词

C-k 移除从光标到“行尾”间的字符

M-k 删除光标后面的句子
C-x 删除光标前面的句子

M-j(自定义) 剪切行

C-x C-o 删除空白行

复制,剪切，粘贴

C-x C-f 寻找文件。
C-x C-s 保存文件。
C-x C-b 列出缓冲区。
C-x C-c 离开 Emacs。
C-x u 撤销。
C-j(自定义) 复制行

标记

M-h 标记段落
C-x C-p 标记页面
C-x h 标记buffer
C-M-h 标记函数

C-x w . 高亮符号
C-x w r 取消高亮符号

交换

C-t 交换字符
M-t 交换单词
C-x C-t 交换两个文本行的位置(当前行和上一行）

列编辑

C-x r M-w 复制
C-x r k 复制并删除
C-x r d 删除(不留空白)
C-x r c 删除(留空白), clear-rectangle
C-x r o 插入空白的方格
C-x r y 粘贴(注意: 只需要把光标放在第一行即可)
C-x r t 替换/插入

C-x r r 选定内容复制到寄存器(记住输入的寄存器名字)
C-x r i 复制寄存器内容

C-x r N [M-x: set-mark-command] 插入递增的数字
插入字母(从A开始)
第一步: C-u [M-x: set-mark-command]
第二步: C-x r N [M-x: rectangle-number-lines]
第三步: 把%2d 改为 %c
第四步: 输入字符asscii码(A:65, a:97)

C-x : 整行移动
第一步:选择区域
第二步: C-x
第三步: 左右方向键移动

查找

字符查找

C-s: 向前查找
C-r: 向后查找
C-s C-s: 恢复上次查找
C-r C-r: 恢复上次查找
进入C-s后的命令:
C-s, M-n(M-p): 从查找队列中恢复历史查找
M-c: 快速切换大小写敏感
C-j: 查询字符串中输入换行
C-w: 使用光标后面的单词查找(C-w可继续向后增加单词)
M-s C-e: 使用光标到行尾的内容查找
M-%: 交互替换
M-e: 编辑查找字符串 (使用C-s,C-r恢复), 可使用C-y把光标后内容复制到查找区
M-y: 把kill环中的内容复制到查找区

文件查找

文件

C-x C-v　在当前缓冲区打开另一个文件
C-x C-w　另存为文件

替换

M-x replace-string RET oldstring RET newstring RET
M-% 响应式替换

代码

C-c C-c 批量注释选中代码
C-u C-c C-c 取消注释
M-. 查找定义

扩展命令

C-x 字符扩展。 C-x 之后输入另一个字符或者组合键。
M-x 命令名扩展。M-x 之后输入一个命令名。

样式

C–: 缩小字体
c-+: 增大字体

mode

dired mode

M-x dired 或者 C-x d： 进入
C-x C-f 创建文件
s ;;按照时间排序

查看文件内容

v 只读模式查看文件
q 退出只读模式并关闭文件
f 进入文本编辑模式

文件操作

d 加上待删除标记
u 取消待删除标记,其实也可取消其他标记
x 删除标记的所有文件

#所有自动保存文件加上待删除标记
~ 给所有备份文件加上待删除标记
D 大D立即删除指定文件
C 大C用于复制该文件到一个目录中
R 重命名文件，也可用于移动文件
Z (dired-do-compress)压缩与解压缩
= 比较改文件与另一个待会输入的文件，类同于UNIX中的diff命令
